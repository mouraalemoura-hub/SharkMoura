# Contrato de Dados para Opções

## Estrutura Geral

Define contratos para seleção de opções, incluindo código, tipo, strike e métricas de risco.

## Campos Principais
- Código da Opção, Tipo (CALL/PUT), Strike, Vencimento.
- Prêmio, Delta, Open Interest.

## Validações

Garante códigos em maiúsculo, preços positivos e consistências entre campos.
