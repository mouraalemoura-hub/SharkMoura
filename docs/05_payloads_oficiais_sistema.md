# Payloads Oficiais do Sistema

## Visão Geral

Os payloads são estruturas de dados padronizadas para comunicação entre componentes do sistema SharkMoura 2.0.

## Payloads Principais
- AssetAnalysisPayload: Análise completa do ativo.
- OptionsSelectionPayload: Seleção de opções.
- FinalSystemPayload: Consolidação final.

## Estrutura

Cada payload herda de BaseSchema, com validações e serialização definidas.
